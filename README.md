 # Hair Salon Booking (MVP)

##  Опис проєкту

Це MVP веб-системи **онлайн-бронювання перукарні**, створений у межах навчальної практики.  
Проєкт реалізує базову логіку:

- вибір послуги;
- перегляд доступних таймслотів;
- створення бронювання;
- перевірку місткості слоту (capacity = 3);
- перегляд бронювань у демо-режимі адміністратора/майстра.

Фронтенд написано на **HTML + CSS + JavaScript**,  
бекенд реалізовано у вигляді **mock API** на основі `json-server`.

Проєкт відповідає варіанту:

- **домен:** перукарня;  
- **місткість слоту:** 3;  
- **NFR-акцент:** performance-first.
