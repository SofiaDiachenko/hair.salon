 MVP Документація: Hair Salon Booking System 
 Мета MVPСтворити мінімально життєздатний прототип (MVP) системи онлайн-бронювання для перукарні ("Салон Ваш Стиль").
 Основні функції, реалізовані в MVP:
 Перегляд доступних послуг.
 Перегляд вільних часових слотів (capacity $\le 3$ бронювань на слот).
 Створення бронювання (імітація POST /bookings).
 Демонстрація UX-станів: Loading, Empty, Success, Error.
 NFR-акцент: Performance-first
 Прототип фокусується на мінімальному розмірі відповідей та імітації кешування для забезпечення високої швидкості взаємодії.
Структура ПроєктуПроєкт розділено на логічні частини: документація, фронтенд-прототип та мок-дані API.hair.salon/
